# YUJOO-OS
- This Project is studying the operating system from 2016.04.08 to 2017 2.1
- 64비트 멀티코어 OS 원리와 구조1(한빛미디어)책을 이해하고 소스를 단계별로 실행시켜 단기간에 간단한 운영체제를 만들어보는 것을 목적으로 한다.
